# Screenshots

Drop screenshots and demo videos in this folder. The top-level README references:

- `demo.png` — hero image, displayed at the top of the README. A wide screenshot of the full 3-pane app mid-conversation works well (1600×900 or wider). A short animated GIF or APNG named `demo.png` also works.

Suggestions for additional shots you might want to add and link from the README:

- `upload.png` — the drag-and-drop panel mid-drop, with a file hovering and the dashed border highlighted.
- `chat.png` — a Q&A exchange showing inline `[1]` `[2]` source pills under an answer.
- `sources.png` — the right-hand sources panel with the green/amber/gray score badges visible.
- `architecture.png` — exported PNG of the mermaid diagram from `docs/architecture.md` if you want to embed it as an image instead.

Any image you drop here is served as-is to anyone browsing the repo on GitHub.
